# Тренажёр ГБОУ Школа № 2200

Статический сайт-тренажёр для подготовки к ликвидации академической задолженности.
Хостинг — бесплатный, на GitHub Pages.

## Адрес сайта
https://pavelneumoin.github.io/2200/

## Как выложить обновление
Самый простой способ — дважды кликнуть по файлу **deploy.cmd**.

Или вручную в этой папке:
```
git add -A
git commit -m "обновление"
git push
```
Через ~1 минуту сайт обновится сам.
