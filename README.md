# Тренажёр ГБОУ Школа № 2200

Веб-тренажёр для подготовки к **ликвидации академической задолженности**.
**31 предмет · 2–11 классы · 24 775 заданий.** Статический сайт без сервера, работает офлайн (PWA, можно «установить» на телефон).

## Адрес
https://pavelneumoin.github.io/2200/

## Возможности
- Режимы: **Тренировка** (17 вопросов / 45 мин), **Блиц** (12 быстрых), **Работа над ошибками** (повтор того, где ошибся).
- 3 типа вопросов: выбор ответа, короткий ответ, сопоставление. Математика — формулы через **KaTeX**.
- **Личный кабинет:** ученик (статистика, графики, достижения, умные рекомендации), родитель, учитель (демо-кабинет класса).
- Оценка 2–5, разбор ответов, фильтр «только ошибки».
- Всё хранится **только на устройстве** (localStorage), ничего не отправляется и не публикуется.
- Клавиатура в тесте (1–8 — ответ, ← → — переходы, Enter — далее), продолжение незавершённого теста.

## Структура
- `index.html` — оболочка SPA
- `css/ds.css` — дизайн-система школы (токены), `css/app.css` — стили приложения
- `js/app.js` — экраны + роутер · `js/engine.js` — генерация и оценка · `js/store.js` — прогресс · `js/icons.js` — иконки Lucide
- `data/index.json` — каталог предметов · `data/q/<предмет>-<класс>.json` — вопросы
- `assets/` — шрифты (Manrope, PT Serif), KaTeX, иконки приложения
- `sw.js`, `manifest.webmanifest` — офлайн и установка

## Обновить вопросы / пересобрать данные
Данные нарезаются из банка скриптом `_testy-2200-tools/build_site.js`:
```
node build_site.js
```
(он же собирает `css/ds.css` из дизайн-системы и иконки).

## Выложить обновление
Двойной клик по **deploy.cmd** (или `git add -A && git commit -m "..." && git push`). Сайт обновится через ~1 минуту.
